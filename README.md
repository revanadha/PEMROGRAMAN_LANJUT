# Pemeriksa Tahun Kabisat (Java)

Proyek ini adalah program sederhana yang ditulis dalam **Java** untuk menentukan apakah suatu tahun yang dimasukkan oleh pengguna adalah **tahun kabisat** atau bukan.

## 🚀 Cara Menjalankan

### 1. Prasyarat

Pastikan Anda telah menginstal **Java Development Kit (JDK)** di sistem Anda.

### 2. Kompilasi

Simpan kode di atas dengan nama `LeapYearChecker.java`. Buka terminal atau command prompt, navigasi ke direktori file, dan kompilasi file Java:

```bash
javac LeapYearChecker.java
